# Volume Discount for WooCommerce
### PET project, by Chinmay.

## Instructions:
1. Upload the `volume-discount` folder to the `/wp-content/plugins/` directory.    
2. Activate the plugin through the `Plugins` screen in WordPress.    
3. Go to the plugin settings page (`Settings` > `Volume Discount`) to configure discount matrix.
4. Futher Configuration:   
4.1. To enable discounts per product, edit a product and locate the `Volume Discount` panel.    
4.2. To enable discounts per category, edit a category and locate the `Volume Discount` panel.    
4.3. To enable discounts per user, edit a user and locate the `Volume Discount` option.    
5. To show the volume discount widget, go to any page and add the `[vd_cart_discounts]` shortcode.
